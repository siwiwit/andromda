Create UserService

This page provides directions to create the UserService using MagicDraw. The model is shown below for your reference.

{mosimage}

{mosimage}

We will create UserService in a package named org.andromda.timetracker.service. When finished, the containment tree in MagicDraw will look like the screen shot on the right.

  1. In the TimeTracker model create a package called service under org.andromda.timetracker.
  2. In the Containment Tree on the left, right-click on the service package and select New Diagram > Class Diagram. Type in Services as the name of this diagram and click OK. The diagram is created and opened in a new window.
  3. Add a new class to the diagram and name it UserService.
  4. Add the stereotype called Service to UserService.
  5. Let's add the getAllUsers() method to UserService. Right-click on the UserService class and select Insert New Operation. A new operation is inserted with the specification "+unnamed1()". Write over this specification with this new one: "+getAllUsers()". Do not specify the return value right now because MagicDraw gets confused with the UserVO[] syntax. After typing the operation click outside the class exit the edit mode.
  6. Now double-click the operation to open the Operation Specification dialog box.
  7. Choose UserVO[] in the Return Type drop-down and then click OK.
  8. Make sure your class diagram matches the one shown above.
  9. Make sure that the MagicDraw containment tree matches the screen shot shown above.
  10. Save the model by selecting File > Save Project.

We are now ready to generate code for the UserService. Please go back to the main tutorial page and continue from where you left off.
