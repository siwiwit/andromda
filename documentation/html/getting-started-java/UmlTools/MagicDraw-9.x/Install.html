MagicDraw 9.5 Installation

This page provides directions to download and install MagicDraw 9.5, a tool for creating UML models. Please note that MagicDraw 9.5 supports UML 1.4 and XMI 1.2. For UML 2 and EMF support, please look at MagicDraw 15.5 installation instructions.

Please note that MagicDraw is a commercial tool but a free Community Edition is available. The Community Edition is not open-source. Here's an excerpt from the Community Edition readme file: "MagicDraw Community edition is a free edition for developers working on non-commercial projects." Also note that the AndroMDA project is not affiliated with MagicDraw in any way.

Follow the instructions below to install MagicDraw 9.5 Community Edition.

  1. Go to the MagicDraw ftp site
  2. Download and install MagicDraw 9.5 for your machine. MD_UML_95_sp1_CE_win.exe is the preferred installer for Windows machines.

Set up two path variables in MagicDraw specifying the location of your Maven repositories.

  1. Start MagicDraw.
  2. Choose Options > Environment from the main menu.
  3. Click on Path Variables.
  4. Click on Add in the Path Variables panel.
  5. Enter maven1.repository in the Name field.
  6. If you have Maven 1 installed, enter the location of your Maven 1 repository in the Value field, e.g. C:\Documents and Settings\your user name\.maven\repository. If you do not have Maven 1 installed, enter any path here, e.g. C:\. This is just to avoid MagicDraw asking for your Maven 1 repository location when opening models.
  7. Click OK.
  8. Click on Add in the Path Variables panel.
  9. Enter maven2.repository in the Name field.
  10. Enter C:\Documents and Settings\your user name\.m2\repository in the Value field.
  11. Click OK twice.
  12. Close MagicDraw.
