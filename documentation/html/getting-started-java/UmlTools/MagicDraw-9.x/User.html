Create User Entity

This page provides directions to create the User entity using MagicDraw. The model is shown below for your reference.

{mosimage}

{mosimage}

We will create User in a package named org.andromda.timetracker.domain. When finished, the containment tree in MagicDraw will look like the screen shot on the right.

  1. In the TimeTracker model create a package called domain under org.andromda.timetracker.
  2. In the Containment Tree on the left, right-click on the domain package and select New Diagram > Class Diagram. Type in "Domain Objects" as the name of this diagram and click OK. The diagram is created and opened in a new window.
  3. Add a new class to the diagram and name it User.
  4. Add the stereotype called Entity to User.
  5. Add the 3 attributes to User.
  6. Double click on the class to open the class specification.
  7. Click on the Tagged Values tab and then click on the tagged value named @andromda.persistence.table. This tagged value allows you to override the default table name for the entity. Click the Create Value button. Enter the value USERS in the right panel. Click OK to add this tagged value to the User entity.
  8. Note that the tagged value shows up in the diagram. To suppress it, right-click on the entity and choose Presentation Options, then deselect Show Tagged Values.
  9. Make sure your class diagram matches the one shown above.
  10. Make sure that the MagicDraw containment tree matches the screen shot shown above.
  11. Save the model by selecting File > Save Project.

Now go back to the main tutorial page and continue from where you left off.
