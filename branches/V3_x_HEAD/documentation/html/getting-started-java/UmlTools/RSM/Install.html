RSM Installation

This page provides directions to configure Rational Software Modeler / Architect, tools for creating UML models. AndroMDA support these tools thanks to their ability to export models to EMF / UML2 files. During the rest of this tutorial, RSM will designate both Rational Software Modeler and Rational Software Architect

Set up a path variables in RSM specifying the location of your Maven 2 repository. Models generated by AndroMDApp contain references to this variables.

  1. Start RSM.
  2. Choose Window > Preferences from the main menu.
  3. Click on Modeling, then Path maps.
  4. Click on Create... in the Path Map panel.
  5. Enter m2repository in the Name field.
  6. Enter the location of your Maven 2 repository in the Location field, e.g.C:\Documents and Settings\your user name\.m2\repository
  7. Click OK twice.
  8. Close RSM.
