ArgoUML Installation

This page provides directions to download and install ArgoUML 0.24, a tool for creating UML models. Please note that versions prior to 0.20 output only UML 1.3 and are not compatible with AndroMDA.

ArgoUML is an open source UML modeling tool. The AndroMDA project is not affiliated with ArgoUML in any way.

Follow the instructions below to install ArgoUML.

  1. Go to the ArgoUML release site
  2. Download and install ArgoUML for your machine.
  3. Download an ArgoUML compatible version of the AndroMDA profile from the ArgoUML CVS and save it someplace where you can reuse it.
