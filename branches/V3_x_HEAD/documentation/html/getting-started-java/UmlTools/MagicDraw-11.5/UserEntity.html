Create User Entity

This page provides directions to create the User entity using MagicDraw 11.5.

We will create User in a package named org.andromda.timetracker.domain.

{mosimage}

  1. In the TimeTracker model create a package called domain under org.andromda.timetracker.
  2. In the Containment Tree on the left, right-click on the domain package and select New Diagram > Class Diagram. The diagram is created and opened in a new window. Type in Domain Objects as the name of this diagram and press Enter.
  3. Add a new class to the diagram and name it User.
  4. Add the stereotype called Entity (The one from AndroMDA Profile, starting with an 'E' and not the one from UML2 Standard Profile, starting with an 'e') to User.
  5. Add the 3 attributes to User.
  6. Make sure your class diagram matches the one shown on the right.
  7. Make sure that the MagicDraw containment tree matches the screen shot shown below.
  8. Save your timetracker model by selecting File > Save Project.
  9. Export it in EMF/UML2 format by selecting File > Export > EMF UML2 XMI. Choose C:/timetracker/mda/src/main/uml as export directory and click Open.

{mosimage}

Now go back to the main tutorial page and continue from where you left off.
