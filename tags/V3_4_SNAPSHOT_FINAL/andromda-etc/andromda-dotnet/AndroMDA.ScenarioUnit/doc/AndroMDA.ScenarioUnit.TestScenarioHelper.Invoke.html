Remarks

this method is called from an NUnit test case and does the following:

  • Calls the method specified by the methodName parameter; passing in all input parameters by using the concrete implementation of IDataProvider configured for each parameter.
  • Captures the actual output returned from the test method and compares it with the expected output using the concrete implementation of IAsserter configured for each return value and out/ref parameter.
