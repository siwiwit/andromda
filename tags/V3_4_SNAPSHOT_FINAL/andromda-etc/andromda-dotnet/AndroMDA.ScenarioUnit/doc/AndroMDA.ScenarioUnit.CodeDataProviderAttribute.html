CodeDataProviderAttribute Class

This is a concrete implementation of the IDataProvider interface. This data provider uses helper methods written in the TestFixture class or a separate class to load input data for test cases.

Remarks

This data provider uses reflection ot execute a helper method on an object reference to load the test data for a specific scenario. This method of loading test data is almost the same as the mechanism used in traditional unit testing where test data is separated out into helper methods. This data provider is useful when used in conjunction with the XMLDataProvider for input data types that cannot be serialized into XML and therefore cannot be loaded using XMlDataProvider.

This class derives from the System.Attribute class. This allows this class to be configured as a data provider for a test method parameter.

The AttributeUsage attribute for this class: [AttributeUsage(AttributeTargets.Class | AttributeTargets.Method | AttributeTargets.Parameter)] ensures that this attribute can be applied at a class, method or at a parameter level.

When this attribute is specified at a TestFixture level, all the test method parameters in that TestFixture are loaded using this data provider unless, if a different data provider is specified at a test method or parameter level. When this data provider is specified as an attribute on the test method, it overrides any data provider configured at the test fixture level. Similarly, when specified at the parameter level, this data provider is used to load that parameter, irrespective of the data providers configured at the test method or test fixture levels.
