Andromda Cruisecontrol build

The scripts directory contains the artifacts to run cruisecontrol for andromda.

backup-ser-files.sh
Save the CC serialized data files
cc-ctl
init.d type script for running CC
cc-timed.xml
Attempt to write a time based schedule for CC
check-mods.sh
Checks for cvs updates
cruisecontrol.xml
Cruisecontrol definition file for galaxy
cruisecontrol-3_x.xml
Cruisecontrol definition file for galaxy version 3
mvn-cc-build.xml
Ant build script to invoke maven 2.
runcc.sh
Runs cruisecontrol directly. Stores the cc pid in pid.
set-env.sh
Set environment variables for the cvs enviroment.
stockholm-cruisecontrol.xml
Cruisecontrol definition file for stockholm
stpcc.sh
Stop cruisecontrols, deprecated for stockholm use cc-ctl or the web jmx interface
test-xsl.sh
Runs xalan with the andromda tailored xsl file in directory xsl, against the log specified as the argument. The logs are in directory ../../logs/andromda-all
test.html
The output from test-xsl.sh
