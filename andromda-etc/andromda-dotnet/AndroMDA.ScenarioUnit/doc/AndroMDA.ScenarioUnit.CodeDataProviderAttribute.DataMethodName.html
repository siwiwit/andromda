CodeDataProviderAttribute.DataMethodName Property

This is an optional property that specifies the name of the helper method to be called by this data provider to load input data.

Remarks

If this property is not specified, this data provider uses a naming convention to identify the helper method to be called.

The naming convention used is: 

[testMethodName]_[parameterName]_data()

The helper method must be a private instance method that has the following signature: 

[c#]
private object [testMethodName]_[parameterName]_data(ParameterInfo pinfo, string methodName, string scenarioName)
