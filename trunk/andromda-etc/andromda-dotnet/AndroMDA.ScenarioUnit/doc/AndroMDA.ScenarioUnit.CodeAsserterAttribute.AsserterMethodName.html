CodeAsserterAttribute.AsserterMethodName Property

This is an optional property to specify the name of the helper method that is called to assert actual output against expected output.

Remarks

If this property is not set, a naiming convention is used to identify the method to be called. The naming convention is: 

[testMethodName_[parameterName]_assert()
Note    In case of return values, there is no "parameterName" part in the naming convention.
