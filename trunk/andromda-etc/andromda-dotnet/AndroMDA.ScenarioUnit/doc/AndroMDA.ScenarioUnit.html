AndroMDA.ScenarioUnit library

AndroMDA.ScenarioUnit Namespace

Namespace hierarchy

Classes

Class Description
CodeAsserterAttribute This is a concrete implementation of the IAsserter interface that uses helper methods to assert actual output from a test case against expected output.
CodeDataProviderAttribute This is a concrete implementation of the IDataProvider interface. This data provider uses helper methods written in the TestFixture class or a separate class to load input data for test cases.
TestScenarioHelper This is the helper class that is used to invoke the functionality provided by the AndroMDA.ScenarioUnit framework.
XMLAsserterAttribute This is a concrete implementation of the IAsserter interface that uses XML files to store and compare expected output with actual output objects serialized into XML.
XMLDataProviderAttribute This is a concrete implementation of the IDataProvider interface. This data provider loads input data stored as serialized XML objects in files.

Interfaces

Interface Description
IAsserter This interface is implemented by all concrete Asserter classes. The implementation of this interface provides the ability to assert that actual output is equal to expected output for a specific test scenario.
IDataProvider This is the interface implemented by all data providers.
