Parameters

pInfo
This is the System.Reflection.ParameterInfo object that contains information about the input parameter to be loaded.
methodName
This is the name of the test method to which the loaded input parameter will be passed. The XML and code-based data providers use this name as a part of of a naming convention to locate the source of input data.
scenarioName
This is the name of the test scenario for which input data needs to be loaded. The XML and code-based concrete data provider implementations use this scenario name as a part of a naming convention to locate the source of input data.
testFixture
This is a reference to an object that contains any helper methods or information that may be needed by a specific data provider. For example, in case of code-based data provider, this object contains the helper method that will return the loaded input parameters.

Return Value

This method returns the loaded input test method parameter as an object.

Remarks

This method is called by TestScenarioHelper on a concrete implementation of this interface for each input parameter of a test method.
