XMLAsserterAttribute.ExpectedOutputDir Property

This property is used in conjunction with the configuration entry 
<add key="TestDataExpectedOutputDir" value="[somePath]" />

This configuration entry specifies the directory in which expected test output is stored as serialized XML. This property is used to specify a sub-directory below the TestDataExpectedOutputDir folder.

Remarks

This property helps organize expected test output data in sub-folders below the TestDataExpectedOutputDir folder.

This property is optional and if it is not set, the methods in this class will look for expected test output in the TestDataExpectedOutputDir folder.
