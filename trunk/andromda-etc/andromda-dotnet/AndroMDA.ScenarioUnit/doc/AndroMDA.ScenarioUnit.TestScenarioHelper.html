Remarks

This class has a single public static method Invoke() that is called from test cases to execute a specific test scenario.

To be able to use this testing framework, the following configuration entries are required in the app.config file of the test harness: 

     <runtime>
        <assemblyBinding xmlns="urn:schemas-microsoft-com:asm.v1">
<dependentAssembly>
<assemblyIdentity name="nunit.framework" publicKeyToken="96d09a1eb7f44a77" culture="Neutral" />
<bindingRedirect oldVersion="2.0.6.0" newVersion="2.2.8.0" />
<bindingRedirect oldVersion="2.1.4.0" newVersion="2.2.8.0" />
<bindingRedirect oldVersion="2.2.0.0" newVersion="2.2.8.0" />
            </dependentAssembly>
</assemblyBinding>
</runtime>

Note    The "NewVersion" value in these entries should be the value of the NUnit version being used in the test harness.

Look at the MoneyServiceSample in the samples folder of the AndroMDA.ScenarioUnit source to see examples of simple usage of this test framework.

To see samples of all possible combinations in which this testing framework can be used, look at the test cases implemented for this framework itself in the AndroMDA.ScenarioUnit.Tests project.
