CodeDataProviderAttribute This is a concrete implementation of the IDataProvider interface. This data provider uses helper methods written in the TestFixture class or a separate class to load input data for test cases.
XMLDataProviderAttribute This is a concrete implementation of the IDataProvider interface. This data provider loads input data stored as serialized XML objects in files.

Remarks

A data provider is used by the TestScenarioHelper class to load the input data for a test scenario and pass it to a test method.

A data provider that implements this interface needs to derive from the Attribute class so that it can be configured to be used to read input data.

The data provider should use the followingAttributeUsage tag:

[AttributeUsage(AttributeTargets.Class | AttributeTargets.Method | AttributeTargets.Parameter)]

This allows a data provider to be configured at a TestFixture, TestMethod, or a parameter level, giving fine grained control to the developers writing unit tests over how test input data is provided for each test case.
